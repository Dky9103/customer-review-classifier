
# 🧠 Customer Review Classifier

A machine learning project that classifies customer reviews into Positive, Negative, or Neutral using Natural Language Processing (NLP) techniques.

## 📁 Files
- `reviews.csv` – Sample review dataset
- `main.py` – Python script for training and testing the model
- `model.pkl` – Saved trained model
- `tfidf.pkl` – Saved TF-IDF vectorizer

## 🚀 How to Run
```bash
pip install pandas numpy scikit-learn nltk
python main.py
```
